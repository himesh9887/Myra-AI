## Notes

- WhatsApp ke liye contact `.env`, `myra_contacts.json`, ya `add whatsapp contact Name Number` se save kar sakte ho.
- Auto-send aur auto-call saved contact ya direct number ke saath best kaam karte hain.
- Screen awareness command current screen ka summary de sakti hai aur reference capture save karti hai.
- File delete aur existing file overwrite pe MYRA confirmation mangegi pehle.
- Voice aur WhatsApp Web features ke liye internet required ho sakta hai.
- Kuch system commands Windows permissions aur installed packages par depend karte hain.
